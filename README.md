Este repositorio reúne una serie de proyectos desarrollados para aplicar los fundamentos de la visualización de datos, combinando el análisis de casos, la implementación de gráficos mediante programación y la selección de representaciones visuales adecuadas según los objetivos de análisis.
Se emplean diferentes conjuntos de datos y escenarios para comprender cómo comunicar información de forma clara, facilitar la exploración de datos y apoyar la toma de decisiones mediante visualizaciones efectivas.

Objetivos:
* Analizar casos de visualización de datos utilizando un framework de análisis.
* Implementar distintos tipos de gráficos mediante programación en Python.
* Seleccionar la codificación visual más adecuada según el tipo de información y la tarea analítica.
* Aplicar principios de comunicación visual para representar datos de manera efectiva.

Contenido del repositorio
Proyecto 1: Análisis de Casos: análisis de escenarios de visualización para identificar tipos de datos, atributos, tareas analíticas y codificaciones visuales utilizando el framework **Qué (What), Por qué (Why) y Cómo (How)**.

Proyecto 2: Implementación de Visualizaciones: desarrollo de diferentes visualizaciones mediante programación utilizando Python. Se implementan mapas de calor, gráficos de torta, gráficos de barras y pequeños múltiples (*small multiples*), comparando su efectividad según el contexto de análisis.

Proyecto 3: Identificación de Tareas de Visualización: aplicación del framework de visualización para identificar tareas analíticas, seleccionar la representación gráfica más adecuada e implementar las visualizaciones justificando cada decisión de diseño.

Se usó: Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook

Metodologías y conceptos aplicados: Visualización de datos, Framework de análisis What – Why – How, Codificación visual, Diseño de gráficos, Análisis exploratorio de datos, Comunicación visual de la información.

Aprendizaje: Este repositorio permitió fortalecer las competencias en visualización de datos desde una perspectiva tanto conceptual como práctica. A través del análisis, diseño e implementación de distintas representaciones gráficas, se desarrolló la capacidad de seleccionar la visualización más adecuada según el tipo de datos, los objetivos del análisis y las necesidades de comunicación, comprendiendo que una buena visualización no solo representa información, sino que facilita su interpretación y apoya la toma de decisiones.
