Este repositorio reúne proyectos enfocados en el análisis y la visualización de datos mediante Python. Incluye el diseño e implementación de diferentes tipos de gráficos, aplicando principios de comunicación visual para facilitar la interpretación de la información.

Objetivos:
* Analizar datos mediante representaciones visuales.
* Desarrollar visualizaciones utilizando Python.
* Seleccionar el tipo de gráfico más adecuado según el contexto de análisis.
* Comunicar información de forma clara y efectiva.

Contenido del repositorio
Proyecto 1. Análisis de Casos: Análisis de diferentes escenarios de visualización para identificar tipos de datos, tareas de análisis y codificaciones visuales utilizando el framework "What – Why – How"

Proyecto 2. Implementación de Visualizaciones: Desarrollo de mapas de calor, gráficos de torta, gráficos de barras y small multiples utilizando Python y librerías especializadas para la visualización de datos.

Proyecto 3. Identificación de Tareas de Visualización: Selección e implementación de visualizaciones en función de los objetivos de análisis, justificando la elección de cada representación gráfica.

Se usó: Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook

Metodologías y conceptos aplicados: Visualización de datos, framework de análisis What – Why – How, codificación visual, diseño de gráficos, análisis exploratorio de datos, comunicación visual de la información.

Resultados obtenidos: Desarrollo de habilidades para analizar datos, diseñar visualizaciones y seleccionar el tipo de gráfico más adecuado según el objetivo del análisis, aplicando buenas prácticas de comunicación visual y herramientas ampliamente utilizadas en análisis de datos.
